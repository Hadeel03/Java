1. What does this program print?

public class Test
{
     public static void main(String[] args)
     {
          System.out.println("39 + 3");
          System.out.println(39 + 3);
     }
}

Answer: 
39 + 3
42

2. What is the compile-time error in this program?

public class Test
{
     public static void main(String[] args)
     {
          System.out.println("Hello", "World!");
     }
}

Answer: 
Too many arguments in system.out.println()
To fix it: "Hello, World!"
